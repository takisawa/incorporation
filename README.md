## 設立の流れ


### 1. 電子定款を作成する

- 定款は紙と電子定款がある。
- 電子定款の場合、定款に貼る収入印紙代４万円が不要となる。
- freeeで作成できそう。


### 2. 登記書類の作成

- 設立登記申請書（申請書+登記すべき事項を記した用紙＋登録免許税貼付台紙）
- 払込証明書
- 印鑑届出書
- 代表社員就任承諾書
- 本店所在地及び資本金決定書
